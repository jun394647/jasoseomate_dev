# 자소서메이트

나의 경험/경력, 저장해둔 우수 자소서, 기업 분석 자료를 RAG로 검색해 자소서 초안을 생성해주는 개인용 도구입니다. 별도 API 키 없이 로컬에 설치된 `claude` CLI(구독 계정)를 헤드리스 모드로 호출해 생성합니다.

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 http://localhost:3000 접속 (자동으로 `/dashboard`로 이동).

사전 조건:
- `claude` CLI가 설치되어 있고 로그인되어 있어야 합니다 (`claude -v`로 확인).
- Node.js 20+.

## 구성

- **내 정보**: 경험/경력/프로젝트를 직접 입력하거나 PDF/DOCX/TXT 파일로 업로드 → 저장 시 자동으로 청크 분할 + 임베딩.
- **우수 자소서**: 합격/참고용 자소서를 기업·직무·문항과 함께 저장 → 생성 시 문체/구성 참고 자료로 검색됨.
- **기업 분석**: 기업별 산업 분석, 인재상 저장.
- **지원 관리**: 기업×직무 단위로 지원 건을 만들고, 문항별로 "AI 초안 생성"을 누르면 관련 경험/우수자소서/기업분석을 검색(RAG)해 프롬프트에 주입한 뒤 `claude -p`로 초안을 생성합니다. 모든 저장/생성 이력은 버전으로 남아 복원할 수 있습니다.
- **대시보드**: 지원 현황, 마감 임박, 최근 작성 활동 요약.

## 데이터 저장 위치

- `data/app.db` — SQLite (모든 데이터). 백업하려면 이 파일만 복사하면 됩니다.
- `.cache/transformers` — 로컬 임베딩 모델 캐시 (최초 실행 시 자동 다운로드, 이후 오프라인 동작).

둘 다 git에 커밋되지 않도록 `.gitignore`에 포함되어 있습니다.

## 참고

- 생성 1회당 `claude` 구독의 사용량 한도에 포함됩니다(별도 API 과금 없음). 화면에 표시되는 "참고 사용량"은 참고용 지표입니다.
- 임베딩은 다국어 모델(`Xenova/multilingual-e5-small`)을 Node 프로세스 안에서 직접 실행하므로 별도 서버 설치가 필요 없습니다.
